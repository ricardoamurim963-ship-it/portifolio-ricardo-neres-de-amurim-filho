# 🌍 Engenharia de Contexto e Lógica Física
 
## 📝 Descrição do Projeto
Este projeto explora a aplicação da lógica de programação para modelar e resolver problemas do mundo físico e contextos urbanos. O objetivo é transformar realidades complexas — como o microclima de uma cidade ou a navegação em situações de emergência — em algoritmos estruturados e tomadas de decisão lógicas[cite: 3].
 
O sistema está dividido em dois módulos principais:
1.  **Algoritmo do Microclima Local:** Analisa dados de temperatura, umidade e Índice de Qualidade do Ar (IQA) em pontos específicos de São Paulo (como o Parque Linear Tiquatira), calculando uma "Nota de Conforto Urbano"[cite: 3].
2.  **Simulador de Evacuação Espacial:** Modela um agente que deve atravessar um ambiente doméstico (Quarto, Sala, etc.) enfrentando obstáculos como fumaça e portas trancadas, gerenciando recursos limitados como energia e itens de inventário[cite: 3].
 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.10[cite: 3]
* **Estruturas Lógicas:** Condicionais aninhadas, laços de repetição (`for`, `while`) e estruturas de seleção `match/case`[cite: 3].
* **Ferramentas:** Google Colab[cite: 3].
 
## 📊 Funcionalidades e Regras de Negócio
### Módulo de Microclima
* **Classificação de IQA:** Categoriza a qualidade do ar entre "Boa" e "Muito Ruim"[cite: 3].
* **Cálculo de Conforto:** Aplica penalidades na nota base (10) se a temperatura estiver fora da faixa ideal (20°C - 30°C) ou se a umidade e qualidade do ar apresentarem níveis críticos[cite: 3].
 
### Módulo de Evacuação
* **Gestão de Energia:** Cada movimento ou obstáculo (fumaça) consome a energia do agente. O sistema monitora a falha ou sucesso da missão com base nesse recurso[cite: 3].
* **Lógica de Inventário:** Implementa a necessidade de encontrar uma chave para abrir portas trancadas, demonstrando interdependência entre os locais[cite: 3].
 
## 🧠 Reflexão Crítica e Aprendizados
O desenvolvimento deste projeto permitiu uma compreensão profunda de como a lógica de programação altera a percepção da realidade:
* **Decomposição de Problemas:** A capacidade de dividir ambientes comuns em etapas claras para que o computador possa processar situações complexas[cite: 3].
* **Tomada de Decisão Estratégica:** O uso de estruturas `if/else` no código reflete a importância de avaliar cenários e consequências na vida cotidiana, agindo de forma menos impulsiva e mais analítica[cite: 3].
* **Persistência e Melhoria:** A implementação de loops ensina que a resolução de problemas é um processo contínuo de testes, correções e repetições até o sucesso[cite: 3].
 
## 🔧 Como Executar
1. Abra o arquivo `projeto_engenharia_de_contexto_e_logica_fisica.py` no Google Colab[cite: 3].
2. Execute a **Parte 1** para visualizar o relatório de conforto urbano dos pontos registrados[cite: 3].
