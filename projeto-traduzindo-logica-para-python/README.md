# 🐍 Projeto: Traduzindo Lógica para Python
 
## 📝 Descrição do Projeto
Este repositório contém uma coleção de algoritmos desenvolvidos em Python que demonstram a transposição de lógica de programação e pseudocódigo para aplicações funcionais. O foco central é a implementação de estruturas de controle de fluxo, laços de repetição e manipulação de tipos de dados para resolver problemas do cotidiano acadêmico e financeiro.
 
O projeto aborda quatro domínios principais:
* **Processamento de Vendas:** Cálculo de subtotais com aplicação de descontos progressivos (5% e 10%) e validação de entradas de estoque.
* **Análise Climática:** Monitoramento de temperaturas semanais com geração automática de alertas para condições extremas e cálculo de médias.
* **Gestão Acadêmica:** Sistema de notas para turmas, classificando alunos entre Aprovação, Recuperação e Reprovação com base em médias aritméticas.
* **Simulador Financeiro:** Cálculo de juros compostos em investimentos com aportes variáveis e rastreamento de metas financeiras.
 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Ferramentas:** Google Colab
* **Paradigma:** Programação Estruturada e Funcional
 
## 📊 Funcionalidades e Regras de Negócio
O sistema foi construído utilizando conceitos fundamentais de Python:
* **Tipagem Dinâmica:** Conversão explícita de entradas (`input`) para tipos numéricos (`float` e `int`) para garantir a integridade dos cálculos.
* **Intervalos Meio-Abertos:** Uso da sintaxe `range(1, limite + 1)` para replicar o comportamento inclusivo de laços de repetição de pseudocódigos tradicionais.
* **Alertas Operacionais:** Implementação de flags lógicas para detectar e sinalizar situações perigosas ou metas atingidas (ex: meta de R$ 10k em investimentos).
 
## 🔧 Como Executar
1. Acesse o arquivo `projeto-traduzindo-logica-para-python.py` ou utilize o link do **Google Colab**.
2. Escolha uma das funções para testar:
   - `processar_vendas()`
   - `analisar_clima()`
   - `sistema_notas_turma()`
   - `simulador_poupanca()`
3. Interaja através do terminal inserindo os dados solicitados conforme o script.
 
## 🧠 Reflexões sobre a Linguagem
Durante o desenvolvimento, foram validados comportamentos específicos do Python:
* **Manipulação de Strings:** A importância do casting (`float()` ou `int()`), pois o Python interpreta entradas padrão como texto, o que impediria operações matemáticas.
* **Lógica de Parada:** A compreensão de que no `range(início, fim)`, o valor final é exclusivo, exigindo o ajuste manual para cobrir todo o período desejado.
